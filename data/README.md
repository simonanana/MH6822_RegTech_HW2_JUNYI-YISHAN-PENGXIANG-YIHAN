# Data Folder

This folder holds the reproducible input data for the HW2 engine.

- `raw/trades.json`: immutable case-file source facts.
- `processed/trades.json`: normalized runtime data generated by `tools/prepare_data.py`.
- `product_definitions/`: ANNA-DSB Product-Definitions, used for UPI template and codeset lookup.
- `data_manifest.json`: generated audit manifest with raw/processed file hashes and product-definition version metadata.

Run this from `HW2_solution` before rebuilding outputs:

```bash
python tools/prepare_data.py
python tools/data_audit.py
python run_compliance_check.py --input data/processed/trades.json --regimes CFTC,EMIR
```

`CFTC,MAS` can be run as an alternative Singapore-nexus view.

If `product_definitions/` is missing and no local copy is available, run:

```bash
python tools/prepare_data.py --download
```
